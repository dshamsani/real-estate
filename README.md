# Real Estate Rental Platform

## Installation

```bash
pnpm install
```

## Development

Run both frontend and backend:
```bash
pnpm start
```

Run frontend only:
```bash
pnpm dev
```

## Build

```bash
pnpm build
```

## Preview Production Build

```bash
pnpm preview
```

## Tech Stack

- **Frontend:** React 18 + TypeScript + Vite
- **Backend:** Node.js + Express
- **Database:** MongoDB
- **Package Manager:** pnpm
